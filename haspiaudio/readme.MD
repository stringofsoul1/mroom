# Home Assistant Add-on: HassPiAudio

Raspberry DLNA Server and DLNA Renderer on local 3.5mm Headfone out.
Also Snapcast Server & Client for MultiRoomAudio!

Based on: badaix/snapcast and hzeller/gmrender-resurrect

DLNA Server -> DLNA Render -> Snapcast Server -> Snapcast -> Audio Out
